# 佩奇（原「宝贝成长记录」）

一个帮助家长记录多个孩子 **基础信息、身高体重成长曲线、每日乖不乖状态** 的本地 Android 应用。整体视觉和交互风格尽量贴近 **Apple / iOS 风格**：浅色、留白多、大标题、圆角卡片。

## 功能简介

- **多孩子管理**
  - 录入多个孩子的基本信息（目前支持：姓名、头像、性别、生日）。
  - 首页以 Apple 风格卡片列表展示孩子，左侧为圆形头像（无头像时显示名字首字母），右侧为名字和提示文案。
  - 卡片底部提供三个清晰入口：**成长记录**、**乖不乖日历**、**编辑资料**，查看数据与编辑信息彻底分离。
- **成长数据记录 + 成长影像**
  - 为指定孩子按日期记录身高、体重及备注。
  - 录入身高/体重时，可同时从 **相册选择照片** 或 **拍照** 作为当次记录的成长照片，记录列表中支持小图预览。
  - 根据记录生成简单的成长曲线：支持 **身高+体重组合视图**、**仅身高**、**仅体重** 三种模式切换，并带有背景网格线，便于阅读。
  - 内置「成长影像」页面：自动按时间轴轮播所有带照片的记录，形成简单的成长幻灯片回顾。
- **每日乖不乖状态（日历视图）**
  - 为指定孩子记录每天的「乖 / 一般 / 不乖」状态。
  - 在日历中以不同颜色的圆点高亮当天状态，可左右切换月份查看。

## 技术栈

- **语言**：Kotlin
- **UI**：Jetpack Compose + Material3，自定义 Apple 风格主题
- **导航**：Navigation Compose
- **数据存储**：Room（SQLite），全部数据仅保存在本机
- **架构**：ViewModel + Coroutines + Flow

## 代码结构概览

- `app/src/main/java/com/example/kids/`
  - `MainActivity.kt`：应用入口，设置 Compose 内容。
  - `KidsApp.kt`：`Application` 子类（预留全局初始化入口）。
  - `ui/`
    - `KidsNavHost.kt`：导航图与路由定义。
    - `theme/Color.kt`、`theme/Theme.kt`：Apple 风格配色与主题。
    - `screens/`
      - `KidListScreen.kt`：孩子列表页（大标题 + 卡片列表 + 添加按钮 + 头像 + 查看/编辑分离按钮）。
      - `KidDetailScreen.kt`：孩子详情编辑页（支持编辑名字、头像、性别、生日，并从详情页进入成长记录/乖不乖日历）。
    - `kid/`
      - `KidListViewModel.kt`：孩子列表状态管理。
      - `KidDetailViewModel.kt`：孩子详情状态管理与保存逻辑。
  - `data/`
    - `model/`：`KidEntity`、`GrowthRecordEntity`、`MoodRecordEntity`
    - `dao/`：`KidDao`、`GrowthRecordDao`、`MoodRecordDao`
    - `db/`：`KidsDatabase`（RoomDatabase）、`Converters`（LocalDate 转换）
    - `repository/`：`KidRepository` 等。

## 在本机编译与运行

> 建议使用 **Android Studio Electric Eel 及以上版本**，JDK 17。

1. **克隆或下载代码**
   - 将本项目放到本地任意目录，例如：`~/Codes/claude-projects/Kids`。
2. **用 Android Studio 打开项目**
   - 打开 Android Studio，选择 `File -> Open...`，选中 `Kids` 目录。
   - 等待 Gradle Sync 完成（右下角进度条结束，`Build` 窗口无错误）。
3. **选择运行配置**
   - 顶部运行配置选择 `app`。
   - 连接真机（推荐）或创建模拟器（API 24+）。
4. **编译 & 运行**
   - 点击工具栏的绿色 ▶ 按钮，或菜单 `Run -> Run 'app'`。
   - 首次会自动构建 `assembleDebug`，成功后会安装到设备上。

## 使用说明（当前实现部分）

1. 打开 App，进入首页「宝贝」列表。
2. 点击「添加宝贝」，进入详情页，输入名字、选择性别、填写生日并设置头像（相册 / 拍照）后点击「保存」。
3. 返回首页即可看到以卡片方式展示的孩子信息，卡片左侧是头像，右侧是名字和提示文字。
4. 点击卡片上的「编辑资料」按钮，可随时修改该宝贝的基础信息。
5. 在首页卡片或详情页中：
   - 点击「成长记录」：为该宝贝按日期记录身高、体重及备注；每条记录可关联一张照片；顶部可切换查看身高曲线、体重曲线或组合视图，并可在右上角进入「成长影像」。
   - 点击「乖不乖日历」：为该宝贝在日历中选择每天的「乖 / 一般 / 不乖」状态，不同状态以不同颜色圆点展示。
